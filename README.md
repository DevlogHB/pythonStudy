### 파이썬 문법 연습

---

##### py_variable

- 변수 문법

##### py_dataType

- 데이터 타입

##### py_logicalFlow

- 보물 찾기 게임(조건문)

##### py_list

- 가위바위보 게임(리스트)

##### py_loop

- 암호 생성 프로그램(반복문)

#### py_dict

- 경매 프로그램(딕셔너리)

#### py_function

- 암/복호화 프로그램 (함수)

#### py_returnFunction

- 계산기 프로그램 (리턴 함수)

#### py_scope

- 랜덤 숫자 맞추기 프로그램 (스코프)
